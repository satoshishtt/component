押さえるポイント

ここでは各自が今回のアプリをもっと成長させたいと思った時に押さえるべきポイントを紹介します。

1.<%= title %>は最終的に表示する時に設定した値に変換され表示する

2.HTMLはタグを入れ子にするのが基本構造である

3.<script>タグをHTMLの中に記述することでJavaScriptを使用できる


解説1




具体的にはindex.jsで記述指定した以下のtitleに設定した値が変換されています。

router.get('/', function(req, res, next) {
res.render('index', { title: 'chat' });
});

解説2


<!DOCTYPE html> <html> <head> <title>chat</title> </head> <body> <div>lesson1</div> <div>lesson2</div> </body> </html>

HTMLを１番上の階層として２番目の階層にHEADとBODYがいます。
更に３番目の階層にTITLEとDIVが存在したツリー構造になっていることが
わかります。

このようにHTMLはタグが入れ子になりで会社組織のような階層構造で作られています。

解説3

$('button').on('click', function() {
