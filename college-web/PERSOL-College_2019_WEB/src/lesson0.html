HTML

HTMLは、画像や文字をブラウザに表示したり、URLのリンクでページを移動できます。
JavaScriptを学習する上でHTMLは一緒に覚えて行くことになります。
ここでは基本的な構造を押さえて起きましょう。解説１を参照


押さえるポイント

ここでは各自が今回のアプリをもっと発展させたいと思った時に押さえるポイントを理解します。

1.htmlに記述した<%= title %>は最終的に表示する時に設定した値に変換されます。
具体的にはサーバー処理であるindex.jsで記述した以下のtitleの値
'chat'が変換されブラウザに表示されています。
---------------------------------------------
index.jsを抜粋
router.get('/', function(req, res, next) {
  res.render('index', { title: 'chat' });
});
---------------------------------------------
※htmlなのに拡張子が「ejs」である理由がこのサーバーサイドから情報を受け取る為です。

2.<script>タグをHTMLの中に記述することでJavaScriptで処理を行うことができる
---------------------------------------------
<script>
$('button').on('click', function() {
  // ボタンを押された時に動作したい処理を書く
});
</script>
---------------------------------------------

解説1


HTMLの基本構造
<!DOCTYPE html> <html> <head> <title>chat</title> </head> <body> <h1>chat</h1> <p>Welcome to chat</p> </body> </html>

<html>を１番上の階層として２番目の階層に<head>と<body>がいます。
更に３番目の階層に<title>と<h1>などが存在したツリー構造になっています

このように<html>はタグが入れ子になりで組織のような階層構造が形成されます。
