まずは変数の宣言とJSONを学びましょう。

変数の宣言

プログラミングではデータを保持するために変数を使用します。
数学のxやyのように、値を代入する箱のようなもので
変数には名前をつけて値を保持しておけます。
JavaScriptでは「var」を変数の前に記述し宣言するが基本です。
記述例：
var test = "パーソル";
これはtestの変数にパーソルという文字列を格納しています。

データ型
変数にはデータ型が存在します。
JavaScriptでは以下の型を基本的に使います。
・数値
・文字列
・論理型
・配列
・オブジェクト
上記以外に特殊な型としてundefinedがあります。
undefinedは変数宣言したけど入れ物に代入していない場合です
 記述例：
var test;

配列型を作成

配列は同一の型のデータの集まりです。 配列型の宣言の仕方は
var ary = new Array();で記述します。
例えば果物情報を配列として中に入れる場合は
ary[0] = "みかん";
ary[1] = "リンゴ";
と記述します。
配列はメソッドでも追加できary.push("バナナ");で順番に格納できます。
console.log();を使用して変数の中身を実際にみましょう。解説を参照


オブジェクト型を作成

オブジェクトは配列と似てますが、名前付きでデータを格納できます。 オブジェクト型の宣言の仕方は
var obj = new Object();で記述します。
情報を中に入れる(プロパティを追加する)場合は
obj["fruits1"] = "みかん";
obj["fruits2"] = "リンゴ";
と記述します。
オブジェクトは[]を使用しなくてもobj.fruits3 = "バナナ";でも記述できます。
console.log();を使用して変数の中身を実際にみましょう。


JSONでの表記

JSONは「JavaScript Object Notation」の略で「ジェイソン」といいます。
配列やオブジェクトをテキストフォーマットで記述できる特徴があります。

配列はnew Array();で宣言していましたが
以下のようにJSONでは角括弧 [ ] とカンマで記述できます
var ary = ["みかん","リンゴ"];

オブジェクトもnew Object();で宣言していましたが
JSONでは中括弧 { }とキー:データで記述できます。
var obj = {"fruits1":"みかん","fruits2":"リンゴ"};
また、JSONはオブジェクトの中に配列など入れ子にすることも出来ます。
var obj = {"fruits":["みかん","リンゴ"]};
