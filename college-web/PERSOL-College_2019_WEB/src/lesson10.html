jQueryとはオープンソースのライブラリで、使用するとJavaScriptが短くシンプルになります。

使用方法

jQueryを使用する方法は２通り
一つはサイトhttps://jquery.com/download/ からダウンロードしファイルを配置する方法。
もう一つは外部のURLアドレスを記述し使用する方法。解説１を参照

記述の仕方

実際にコードを記述するときは$()またはjQuery()と先頭に記述して処理を書きます。

セレクタの使い方

セレクタは指定のHTMLを参照します。
jQueryのセレクタはCSSセレクタが基本となっています。

基本のセレクタ

セレクタ 説明 記述例
.class クラス名を持つ全ての要素を取得 $(".serif-left")
element 要素名を持つ全ての要素を取得 $("button")
#id idを持つ唯一の要素を取得 $("#td_3A")

イベント処理

ブラウザで表示されたページは「ボタンがクリックされた」「マウスをかざした」など
様々な出来事が起こります。これらはプログラミングでは「イベント」と言います。
javascriptではこうしたイベントに応じて実行するコードを記述をしていきます。

代表的なイベント

分類 イベント名 発生タイミング
読み込み load ページ/画像の読み込みが完了した時
unload ほかのページに移動する時
マウス click クリック時
dblclick ダブルクリック時
mouseover マウスポインタが要素に乗った時
mouseout マウスポインタが要素から外れた時
フォーム reset リセットボタンを押した時
submit サブミットボタンを押した時

$.on()メソッドによるイベント処理

$.on()メソッドを使用してイベントを記述することができます。
$('要素').on('イベント名', function(){ 処理 })
実際の記述は解説２のようになります。

複数のイベントの記述の仕方

複数の別々のイベントを設定することもできます。
$('要素').on({
'イベント名':function(){ 処理 },
'イベント名':function(){ 処理 }
});
