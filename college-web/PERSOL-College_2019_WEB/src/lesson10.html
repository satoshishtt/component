jQueryとはオープンソースのライブラリで使用すると短くシンプルなプログラムの記述が可能です。
2006年にJhon Resigが発表し現在は「jQuery Team」が開発と保守を行っています。

使用方法

使用方法は２通りあります。
一つはサイトhttps://jquery.com/download/ からダウンロードしファイルを配置する方法。
もう一つはでURLアドレスで使用する方法。解説１を参照

記述の仕方

実際にコードを記述するときはjQueryの関数やプロパティが入っているオブジェクトの
$()またはjQuery()と記述して処理を書きます。

セレクタの使い方

セレクタは任意のエレメントを、DOM ツリーから選び出すための書式です。
jQueryのセレクタはCSSセレクタが基本となっていますのですぐわかると思います。

セレクタ 説明 記述例

element 要素名を持つ全ての要素を取得 $("span")
#id idを持つ唯一の要素を取得 $("#td_3A")

イベント処理

$.on()メソッドによるイベント処理

$.on()メソッドを使用した基本的な記述方法は以下のようになります。
$('要素').on('イベント名', function(){ 処理 })
イベントリスナの記述方法に似ているので覚えやすいですね。
実際の記述例は解説２のようになります。

複数のイベントの記述の仕方

onメソッドを使って複数の別々のイベントを設定すると以下のようになります。
記述の仕方が少し変わりましたね。
$('要素').on({
'イベント名':function(){ 処理 },
'イベント名':function(){ 処理 }
});
 イベントハンドラの時と比べると少しコードの量が減ったことがわかると思います。
実際の記述例は解説３を見てみましょう。
