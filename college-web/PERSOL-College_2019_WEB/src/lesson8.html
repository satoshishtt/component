CSSのセレクタ

下記のスタイルの記述をみてみましょう。
#tableId2 {
　color : blue ;
}
{}の始めの#tableId2の部分はセレクタといいます。
ここにどの部分に適用したいのかを記述します。
#はIDに対して適用するという意味を持ち、この場合は
tableId2のIdを持つ属性に対しての命令となります。

一括設定と文脈セレクタの記述

下記のスタイルの記述をみてみましょう。
div,p {
　color : blue ;
}
div,pはカンマを使用していますこれは一括設定を指し
両方の要素に対して同じスタイルを適用する場合に用います。

次は似たような記述ですが少し違います。
div p {
　color : blue ;
}
div pは半角スペースを使用していますこれは文脈セレクタと呼び
divの中に囲まれている要素pに対してスタイルを適用する場合に用います。

代表的なセレクタ
種類 記述例 説明
ID #table {color:blue;} tableのidを持つ要素へ適用
書式は#を先頭
CLASS .sample {color:blue;} .sampleの属性クラスを持つ要素へ適用
書式は.を先頭
要素 div {color:blue;} div要素へ適用
書式はタグ名と一緒
属性 input[value="200"] {color:blue;} input要素かつvalueが200の要素に適用
書式は[属性][属性=値]などで記述

解説１

CSSやHTMLは背景や見出しなど文書構造を定義するマークアップ言語です。
JavaScriptはプログラミング言語でマークアップと異なる点は
「順次」「選択」「繰り返し」といった命令文が記述できるのかが違いとなります。
