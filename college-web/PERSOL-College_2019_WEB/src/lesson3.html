条件分岐

次は条件分岐を学びましょう。

条件文

プログラミングでは処理は常に同じではなく、様々に変化します。
条件ごとに処理を変化させたい場合はif文またはSwitch文を使用します。

記述例１：
var a = 10;
if (a > 5) {
　　// ()の条件がtrueの場合に実行される
} else {
　　// ()の条件がfalseの場合に実行される
}
記述例２：
var a = 3;
if (a == 1) {
　　// aの変数の中身が1の場合に実行される
} else if (a == 2) {
　　// aの変数の中身が2の場合に実行される
} else {
　　// aの変数の中身が1と2以外に実行される
}
記述例３：
var a = 3;
switch(a) {
　　case 1:
　　 　　// aの変数の中身が1の場合に実行される
　　 　　break;
　　case 2:
　　 　　// aの変数の中身が2の場合に実行される
　　 　　break;
　　default:
　　 　　// aの変数の中身が1と2以外に実行される
　　 　　break;
}
※caseには定数を指定し条件は記述できない。

比較演算子

判定を行う場合は比較演算子を使用します。
2つの値を比較する演算子で、評価結果で真偽値を返します。
==（等しい）
!=（等しくない）
>（大きい）
>=（以上）
<（小さい）
<=（以下）

同値演算子と等値演算子

演算子には同値演算子(===)と等値演算子(==)があります。
同値演算子はデータ型も含めて比較判定を行います。
例: 1 === '1' 結果: false
等値演算子は異なる型のプリミティブ値を比較する際には
数値に型変換を行なってから比較判定を行います。
例: 1 == '1' 結果: true

算術演算子

数値を計算して比較する際は以下を使用します。
「加算」：「+」 1 + 1 = 2
「減算」：「–」 1 - 1 = 0
「乗算」：「*」 3 * 3 = 9
「除算」：「/」 4 / 2 = 2
「剰余」：「%」 5 % 3 = 2

論理演算子

複数の条件指定して比較する際は以下を使用します。
「and（&&）」
「or（||）」
「not（!）」

記述例：
var num = 5;
if(num >= 0 && num < 10) {
　　// 0以上10より小さい場合に実行される
} else {
　　// 10以上の場合に実行される
}

ループ処理

同じ処理を何度も繰り返す時はループ文で記述します。

for文

for文は以下の3つの部品から構成されます
・カウント変数
・ループの条件
・カウント変数の変化
記述例：
for (var count = 0; count < 10; count++) {
　　// カウント変数が10になったら処理が終了
}

for...in文

オブジェクトにあるプロパティまたは配列を反復する
記述例：
var obj = {"fruits1":"みかん","fruits2":"リンゴ"};
for (var i in obj) {
　　// i = fruits1 obj[i] = みかん
　　// i = fruits2 obj[i] = リンゴ
}

break 文

反復の途中で処理から抜ける
記述例：
var cnt = 0;
for (var i = 0; i < 5; i++) {
　　if (i == 3) {
　　 　　// iが3の時に反復を抜ける
　　 　　break;
　　}
　　cnt += i;
}
alert(cnt);

continue 文

現在の反復は途中終了し、次のループ処理を継続する
記述例：
var cnt = 0;
for (var i = 0; i < 5; i++) {
　　if (i == 2 || i == 3) {
　　// iが2の時と3の時は次の処理をしない
　　continue;
　　}
　　cnt += i;
}
alert(cnt);
