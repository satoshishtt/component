条件分岐

次は条件分岐を学びましょう。

条件文

プログラミング言語は「順次」「選択」「繰り返し」の処理ができます。
処理を「選択」させたい場合はif文を使用します。

記述例１：
var a = 10;
if (a > 5) {
　　// ()の条件がtrueの場合に実行される
　　alert("A");
} else {
　　// ()の条件がfalseの場合に実行される
　　alert("B");
}
記述例２：
var a = 3;
if (a == 1) {
　　// aの変数の中身が1の場合に実行される
　　alert("A");
} else if (a == 2) {
　　// aの変数の中身が2の場合に実行される
　　alert("B");
} else {
　　// aの変数の中身が1と2以外に実行される
　　alert("C");
}

比較演算子

判定を行う場合は比較演算子を使用します。
2つの値を比較する演算子で、評価結果で真偽値を返します。
==（等しい）
!=（等しくない）
>（大きい）
>=（以上）
<（小さい）
<=（以下）

算術演算子

数値を計算して比較する際は以下を使用します。
「加算」：「+」 1 + 1 = 2
「減算」：「–」 1 - 1 = 0
「乗算」：「*」 3 * 3 = 9
「除算」：「/」 4 / 2 = 2
「剰余」：「%」 5 % 3 = 2

論理演算子

複数の条件指定して比較する際は以下を使用します。
「and（&&）」
「or（||）」
「not（!）」

記述例：
var num = 5;
if(num >= 0 && num < 10) {
　　// 0以上10より小さい場合に実行される
　　alert("A");
} else {
　　// 10以上の場合に実行される
　　alert("B");
}

ループ処理

「順次」「選択」「繰り返し」の処理の中で「繰り返し」をさせたい場合はfor文を使用します。

for文

for文は以下の3つの部品から構成されます
・カウント変数の初期値
・ループの終了条件
・カウント変数の増減
for (カウント変数の初期値; ループの終了条件; カウント変数の増減)
記述例：
for (var count = 0; count < 10; count++) {
　　// 10回反復したら処理が終了
}

break 文

反復の途中で処理から抜けることもできます。
記述例：
var cnt = 0;
for (var i = 0; i < 10; i++) {
　　if (i == 5) {
　　 　　// 5回反復で処理を抜ける
　　 　　break;
　　}
　　cnt += i;
}
alert(cnt);
