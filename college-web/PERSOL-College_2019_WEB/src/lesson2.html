関数はJavaScriptだけでなくプログラミングをする上で欠かせない
タスクや値計算などを実行する文の集まりです。

関数の宣言の仕方

関数の定義はfunction キーワードと以下を使用し宣言します。
・関数の名前
・引数
・波括弧
記述例１：引数なし
function sample() {
　　alert("hello!");
}
記述例２：引数あり
function sample2(year, month) {
　　return year + ":" + month;
}

関数の呼び出し

関数の呼び出しは名前に()を付けて呼び出します。
記述例：sample();
実際に記述例１を宣言してそのあとに呼び出してみましょう。



関数宣言の優先順位

もし関数の宣言で同じ名前を使用したらどうなるでしょうか。
関数の宣言は同じ名前が宣言された場合最後に宣言された関数が有効になります。

ここで練習１を実践　時間は５分


無名関数

functionの後ろに名前を付けない関数を無名関数と呼びます。
その場合は変数に代入し使用することができます。
記述例：
var example = function () {
　　alert("thank you!");
};
example();
無名関数を呼び出してみましょう。



練習１


名前が一緒な関数を２つ宣言し後に記述された方が有効かを確かめなさい。
条件：一つ目の関数にはalert(1);の処理を記述し二つ目の関数にはalert(2);を記述する。
